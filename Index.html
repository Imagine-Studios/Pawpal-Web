Política de privacidad

Última actualización: 06/11/2025

Esta política explica cómo TuEstudio trata los datos en la app Nombre del Juego.

Datos que tratamos

  • Datos de uso (p. ej., eventos, tiempo de sesión) para analítica.
  • Identificadores del dispositivo (incluido el ID de publicidad) para anuncios.
  • Datos de diagnóstico (crashes y rendimiento).

Finalidades

  • Proporcionar y mejorar la app.
  • Obtener estadísticas agregadas de uso.
  • Monetización mediante publicidad.

Proveedores

Usamos servicios de terceros que pueden tratar datos en nuestro nombre:

  • Google AdMob (publicidad y medición).
  • Analytics (por ejemplo, Firebase/Unity Analytics) — si aplica.
  • Crashlytics (diagnóstico) — si aplica.

Consulta las políticas de cada proveedor para más detalles.

Consentimiento en la UE (AdMob)

Para usuarios del EEE y Reino Unido solicitamos consentimiento para anuncios personalizados mediante un gestor de consentimiento compatible. Puedes cambiar tu elección desde el menú de Privacidad dentro de la app.

Conservación

Conservamos los datos el tiempo necesario para las finalidades indicadas y según requisitos legales.

Tus derechos

Si resides en el EEE/UK, puedes ejercer acceso, rectificación, oposición o supresión de tus datos. Escríbenos a soporte@tuestudio.com.

Contacto

Responsable: TuEstudio. Email: soporte@tuestudio.com.
